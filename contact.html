Frequently Asked Questions

How do I get access to upload notes?

Contact Raheem directly with your student information. You'll receive manual credentials for login.

How long does note approval take?

Notes are typically reviewed within 24 hours by the admin team.

Can I download notes without an account?

No, you need to be a registered student to access and download notes.

What file formats are supported?

We support PDF, DOC, DOCX, PPT, PPTX, and image files (JPG, PNG).

Is there a file size limit?

Yes, the maximum file size is 50MB per upload.

Can I delete my uploaded notes?

You can delete notes that are still pending approval. Once approved, contact admin for removal.
